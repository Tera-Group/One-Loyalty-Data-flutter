setup

suspend fun setup(context: Context, config: Config, appContext: AppContext, platform: Platform = Platform.ANDROID): RemoteConfig?

Initializes the configuration for the remote service and sets up the app context. This function configures the application environment, sets device-related information, and fetches the remote configuration from a remote service.

Return

the remote configuration fetched from the service, or the default configuration if fetching fails. The returned configuration may have its request timeout updated.

Parameters

context

the Android Context for the current application.

config

the configuration settings required for the remote service, including API keys and default configuration.

appContext

the application-specific context, which includes device information and settings.

platform

the target platform for the configuration, defaulting to Platform.ANDROID. It indicates the platform the application is running on (e.g., Android, iOS).

Throws

Exception

if the configuration setup or fetching remote config encounters an error.
